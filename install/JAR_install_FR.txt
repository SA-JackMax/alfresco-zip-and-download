== Installer les modules sur Alfresco 3.4 ==

1. Arrêter Alfresco
   a. Executer la commande alfresco.sh stop (ou alf_stop.bat selon si vous êtes sous Linux ou Windows)


2. Copier les fichiers JAR
   a. Copier le fichier "share-zip-contents-action.jar" dans le répertoire "/tomcat/shared/lib/" de votre Alfresco (créer le dossier /lib s'il n'existe pas).

   b. Copier le fichier "zip-contents-alfresco-webscript.jar" dans le répertoire "/tomcat/webapps/alfresco/WEB-INF/lib/" de votre Alfresco.


4. Lancer votre Alfresco
   a. Executer la commande alfresco.sh start (ou alf_start.bat selon si vous êtes sous Linux ou Windows)
